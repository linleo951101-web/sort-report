🫧 氣泡排序法

Bubble Sort - 基礎排序算法的原理與應用

算法原理

基本概念

氣泡排序（Bubble Sort）是一種簡單的排序算法，它通過重複走訪要排序的數列，比較相鄰兩個元素的大小，如果順序錯誤就交換它們，直到整個數列排序完成。算法名稱的由來是因為較小的元素會逐漸"浮動"到數列的兩端。

工作原理

  1. 第一輪比較：從數列的第一個元素開始，逐個比較相鄰的兩個元素
  2. 交換操作：如果左邊的元素大於右邊的元素，就交換它們的位置
  3. 完成一輪：經過一輪比較後，最大的元素會"浮動"到數列的末尾
  4. 重複過程：對剩餘的元素重複上述過程，直到整個數列排序完成

算法步驟

algorithm BubbleSort(array)
    n = length(array)
    for i = 0 to n-1
        for j = 0 to n-i-2
            if array[j] > array[j+1]
                swap(array[j], array[j+1])
            end if
        end for
    end for
end algorithm

具體例子

對數列 [5, 2, 8, 1, 9] 進行排序：

初始數列： [5, 2, 8, 1, 9]
第一輪：
比較 5 和 2 → 交換 → [2, 5, 8, 1, 9]
比較 5 和 8 → 無需交換 → [2, 5, 8, 1, 9]
比較 8 和 1 → 交換 → [2, 5, 1, 8, 9]
比較 8 和 9 → 無需交換 → [2, 5, 1, 8, 9]
第二輪：
比較 2 和 5 → 無需交換 → [2, 5, 1, 8, 9]
比較 5 和 1 → 交換 → [2, 1, 5, 8, 9]
比較 5 和 8 → 無需交換 → [2, 1, 5, 8, 9]
第三輪：
比較 2 和 1 → 交換 → [1, 2, 5, 8, 9]
比較 2 和 5 → 無需交換 → [1, 2, 5, 8, 9]
最終結果： [1, 2, 5, 8, 9]

代碼實現

C 語言實現

#include <stdio.h>

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交換操作
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);
    
    bubbleSort(arr, n);
    
    printf("排序後的數組：");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    return 0;
}

C++ 語言實現

#include <iostream>
using namespace std;

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }
        // 優化：如果沒有交換發生，說明已排序
        if (!swapped) break;
    }
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);
    
    bubbleSort(arr, n);
    
    cout << "排序後的數組：";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    return 0;
}

優化版本（提前終止）

如果某一輪比較中没有發生任何交換，說明數列已經排序完成，可以提前終止算法。

void optimizedBubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;
            }
        }
        if (!swapped) {
            break;  // 提前終止
        }
    }
}

性能分析

分析項目 時間複雜度 說明
最佳情況 O(n) 數列已排序，使用優化版本可提前終止
平均情況 O(n²) 典型的隨機排列情況
最壞情況 O(n²) 數列逆序排列，需要最多比較和交換
空間複雜度 O(1) 只需要固定的額外空間
穩定性 穩定 相等元素的相對順序不變

時間複雜度分析

  • 比較次數：第一輪 n-1 次，第二輪 n-2 次，...，最後一輪 1 次，總計 (n-1) + (n-2) + ... + 1 = n(n-1)/2 次
  • 交換次數：最壞情況下等於比較次數，最好情況下為 0 次
  • 總操作數：O(n²) 量級

優缺點分析

✓ 優點

  • 實現簡單，容易理解
  • 無需額外空間，原地排序
  • 穩定排序算法
  • 對小規模數據效率可接受
  • 自適應性好（部分有序數據）

✗ 缺點

  • 時間複雜度高（O(n²)）
  • 大規模數據排序效率低下
  • 比較和交換次數多
  • 不適合實時性要求高的應用
  • 遠不如快速排序和歸併排序高效

應用領域

適用場景

  • 教育目的：作為排序算法的入門學習材料
  • 小規模數據：數據量較小（通常 < 1000）時可用
  • 部分有序數據：使用優化版本，效率相對較好
  • 內存限制嚴格：只需 O(1) 的額外空間
  • 穩定性要求：需要保持相等元素相對順序時

不適用場景

  • 大規模數據排序：性能瓶頸明顯
  • 實時系統：不能滿足性能要求
  • 競賽編程：時間限制會導致超時
  • 生產環境：應使用更高效的算法

總結

氣泡排序是最簡單的排序算法之一，具有以下特點：

  • 算法簡單：代碼容易實現和理解，是學習排序算法的好起點
  • 空間高效：只需 O(1) 的額外空間，適合內存有限的場景
  • 穩定性好：能夠保持相等元素的相對順序
  • 時間效率低：O(n²) 的時間複雜度限制了其應用範圍
  • 實用性有限：在實際應用中應優先選擇快速排序、歸併排序等高效算法

建議：在實際開發中，除非有特殊要求（如教學或數據量非常小），應優先使用標準庫提供的排序函數或選擇時間複雜度為 O(n log n) 的排序算法。
